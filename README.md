# 폴더 자동 생성 어시스턴스

- 프로젝트 시작 시 초기 세팅 작업 시간 줄이고 싶었습니다.
- 요구사항을 입력하면 AI 기반으로 프로젝트 폴더 압축파일을 만들어줍니다.

- 완벽하지 않습니다. 사용하면서 다듬을 예정입니다.

  ## 실행 방법

  ``` bash
  dotnet run --project FolderAssi.Runner -- --ui
  ```
